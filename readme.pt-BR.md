Estrutura de uma entrada de pokemon:

 - int: nº na pokedex global

 - string: nome

 - string: habilidade

 - int: estágio de evolução

 - float: peso

 - int: geração

 - vetor de string: fraquezas

 - vetor de string: resistências

 - vetor de string: imunidades

 - vetor de string: classe de breeding
